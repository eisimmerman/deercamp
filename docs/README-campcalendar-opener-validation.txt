DeerCamp CampCalendar Season Opener Validation Package

Rule: publish verified dates individually. Do not require all 3 season types. Never guess. Third-party sources are leads only; official state wildlife/DNR sources are required before publishing.

Files:
- master-50-state-opener-validation.csv: full 50-state audit table.
- us-state-deer-openers.json: public CampCalendar opener data; includes only currently verified/published states.
- deer-season-openers.json: compatibility copy for older DeerCamp lookup paths; same content as us-state-deer-openers.json.

Current verified/published states in this package: IA, IL, MN, WI.
All other states are explicitly marked pending_official_validation and are not published.
