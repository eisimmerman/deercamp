DeerCamp Facebook Deer Groups Outreach Package

Contents:
1) outreach_files/
   - Airtable import CSV
   - Airtable workbook XLSX
   - posting schedule CSV
   - tracking links CSV
   - caption variants CSV
   - routing map JSON
   - original poster assets (png/jpg)

2) corrected_scannable_posters/
   - Updated poster files with rebuilt QR codes intended to scan more reliably.

3) standalone_qr_codes/
   - Standalone QR images for testing and fallback use.

Recommended files to start with:
- corrected_scannable_posters/fb_core_deercamp_SCANNABLE_QR.png
- corrected_scannable_posters/fb_legacy_stories_SCANNABLE_QR.png
- corrected_scannable_posters/fb_field_memories_SCANNABLE_QR.png
